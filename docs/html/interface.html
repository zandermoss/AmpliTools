class amplitools.interface.Interface(momentum_symbols, polarization_symbols, coefficient_symbols, coefficient_display_map)

An interface between amplitools and jupyter.

Provides input methods converting from simple notebook syntax to MRing and MRational objects, as well as output methods converting MRing and MRational objects to LaTeX expressions (and typesetting them).
